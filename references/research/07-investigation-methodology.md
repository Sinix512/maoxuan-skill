# 毛泽东调查研究方法论系统分析

## 一、《反对本本主义》——调查方法的纲领性文献

### 背景
1930年5月，毛泽东在江西寻乌进行调查期间，为反对当时红军中盛行的教条主义（"本本主义"）思想，写下了《调查工作》（后更名为《反对本本主义》）。该文首次系统阐述了调查研究的理论和方法，是实事求是思想路线的初步阐发。

### 核心论断

| 论断 | 出处 |
|------|------|
| **"没有调查，没有发言权"** | "你对于某个问题没有调查，就停止你对于某个问题的发言权。这不太野蛮了吗？一点也不野蛮。你对那个问题的现实情况和历史情况既然没有调查，不知底里，对于那个问题的发言便一定是瞎说一顿。" |
| **"调查就像'十月怀胎'，解决问题就像'一朝分娩'"** | 调查本身就是解决问题的过程 |
| **"一切结论产生于调查情况的末尾，而不是在它的先头"** | 反对先入为主、主观臆断 |
| **"中国革命斗争的胜利要靠中国同志了解中国情况"** | 独立自主、实事求是的思想雏形 |

后来毛泽东进一步发展为：**"不做正确的调查同样没有发言权。"**

### 七大调查技术

1. **要开调查会作讨论式的调查**
   - 不提出中心问题在会议上经过辩论，不能抽出近于正确的结论
   - 要有争论、有辩论，等意见趋于一致时再记录

2. **调查会到些什么人？**
   - 要找能深切明了社会经济情况的人
   - 老年人（经验丰富，懂得因果）、青年人（思想进步、观察锐利）
   - 范围要广："工人也要，农民也要，商人也要"

3. **开调查会人多好还是人少好？**
   - 取决于调查人的指挥能力
   - 寻乌调查总结会多达50余人参加

4. **要定调查纲目**
   - 事先准备，要有大纲还要有细目
   - 如"商业"下分"寻乌城"，"寻乌城"下分"杂货"，"杂货"再分"布匹""洋货"等

5. **要亲身出马**
   - 从乡政府主席到总书记，"都要亲身从事社会经济的实际调查"
   - "假手于人是不行的"

6. **要深入**
   - 要了解一个地方或一个问题的"底里"
   - "倘若走马看花，如某同志所谓'到处只问一下子'，那便是一辈子也不能了解问题的深处"

7. **要自己做记录**
   - 调查不但要自己当主席，而且要自己做记录
   - "假手于人是不行的"


## 二、《农村调查》的序言和跋——调查方法的系统化总结

### 背景
1941年3-4月，毛泽东将自己1930—1934年间撰写的11篇调查报告汇编成《农村调查》一书，写下序言和跋，作为延安整风22个学习文件之一。

### 核心方法："眼睛向下" + "开调查会"

**第一，眼睛向下，不要只是昂首望天。**
> "没有眼睛向下的兴趣和决心，是一辈子也不会真正懂得中国的事情的。"

**第二，开调查会。**
> "开调查会，是最简单易行又最忠实可靠的方法，我用这个方法得了很大的益处，这是比什么大学还要高明的学校。"

### "甘当小学生"精神（调查态度的灵魂）

> "没有满腔的热忱，没有眼睛向下的决心，没有求知的渴望，没有放下臭架子、甘当小学生的精神，是一定不能做，也一定做不好的。"

具体表现：
- 称调查对象为"我的可敬爱的先生"：干部、农民、秀才、狱吏、商人和钱粮师爷
- 态度要求："恭谨勤劳和采取同志态度"，否则他们就会"知而不言，言而不尽"
- 核心信念：**"群众是真正的英雄，而我们自己则往往是幼稚可笑的"**
- 最终志愿："和全党同志共同一起向群众学习，继续当一个小学生，这就是我的志愿。"

### 阶级分析法——最基本的方法

> "有计划地抓住几个城市、几个乡村，用马克思主义的基本观点，即阶级分析的方法，作几次周密的调查，乃是了解情况的最基本的方法。"

### 对"钦差大臣"作风的批判
> "下车伊始就哇喇哇喇地发议论，这也批评，那也指责，这种人十个有十个要失败。"


## 三、《寻乌调查》——调查方法的经典案例

### 背景与动因（1930年5月）

1. **纠正"左"倾错误**：此前红四军攻占信丰县城时，错误照搬农村没收政策到城市，导致商店关门、商人停市。毛泽东深感自己对城市商业"完全是门外汉"。
2. **弄清富农问题**：土地革命初期，中央对待富农的政策反复变化。
3. **选点逻辑**：寻乌地处闽粤赣三省交界，"明了了这个县的情况，三省交界各县的情况大概相差不远。"——**典型性选点原则**。

### 调查规模
- 历时**20多天**，其中开调查会**十多天**
- 整理出**8万余字**的调查报告（5章39节），是毛泽东一生最长的一篇调研报告
- 走访了**47家商店**和**94家手工业店铺**

### 11位核心调查对象

| 姓名 | 年龄 | 身份 |
|------|------|------|
| 郭友梅 | 59岁 | 杂货店主，曾任县商会会长 |
| 范大明 | 51岁 | 贫农，县苏维埃职员 |
| 赵镜清 | 30岁 | 中农，做过铸铁工、小商贩，当过排长 |
| 刘亮凡 | 27岁 | 曾任县署钱粮办事员，城郊乡苏主席 |
| 李大顺 | 28岁 | 贫农，曾任区苏委员 |
| 刘茂哉 | 50岁 | 老童生，原小地主后降为贫民 |
| 刘星五 | 46岁 | 农民，做过小生意 |
| 钟步嬴 | 23岁 | 梅县师范生，时任区政府主席 |
| 陈倬云 | 39岁 | 缝工、小学教师 |
| 郭清如 | 62岁 | 秀才 |
| 古柏 | — | 寻乌县委书记（组织者兼翻译） |

对象选择特点：年龄23-62岁，涵盖商人、农民、教师、官吏、秀才等各色人物。

### 五大调查纲目

1. 寻乌的政治区划
2. 寻乌的交通
3. **寻乌的商业**（篇幅最长，占近一半）
4. 寻乌的旧有土地关系
5. 寻乌的土地斗争

纲目层层细分，例如："寻乌的商业"→"寻乌城"→"杂货"→"布匹"→"洋布""土布""绸缎""呢绒"。

### 调查会具体操作

在50余人的总结调查会上，毛泽东的操作流程：
1. 摊开几张写满提纲的纸和一个本子
2. 说："今天我提一个问题，大家随便谈一谈"
3. 边听边记，边记边点头，有时插问一两句
4. 允许不同意见争论、互相补充
5. 等意见**接近一致时才记下来**
6. 从头到尾一字一句念一遍，问"是这样吗？"
7. 确认"是这样了"才进入下一个问题
8. 当天先后问了**几百个问题**

### 具体提问示例
- "寻乌做生意的人哪一类最多？"→答"做豆腐、酿水酒的多"
- 追问："寻乌有多少家豆腐店？哪家豆腐做得最好，最容易卖掉？"——**追问到底，不允许"大概"敷衍**

### 交叉验证案例：卖妻鬻子

为验证"卖妻鬻子"现象，毛泽东专门找来**三个不同村**的贫苦农民（刘亮凡、李大顺、梅治平）分别细问：
- 三人各自独立确认所在村子都有卖儿现象
- 估计每百家人中有十家卖过儿子
- **三人独立回答形成交叉验证**
- 据此得出结论：**"旧的社会关系，就是吃人关系！"**

### 主要发现与政策转化

| 领域 | 调查发现 | 政策转化 |
|------|----------|----------|
| 土地问题 | 地主占3.5%人口，拥有30%土地；公田40%；约90%农民仅拥有不足20%土地 | 提出"抽多补少"还要"抽肥补瘦"的原则 |
| 商业问题 | 寻乌城商店多数不雇人或只雇少数店员，都是中小工商业者 | 确立"保护中小商人"政策，小商人和手工业者是"革命的同盟者" |
| 阶级分析 | 县城人口：农民60%，手工业者11%，城市贫民16%，商人仅5% | 为"依靠谁、团结谁、打倒谁"提供科学依据 |
| 富农问题 | 区分"半地主性富农"和"较富裕的自耕农" | 不同对待的细分政策 |
| 文化程度 | 全县约10万人，识字40%，能写信仅3.5%，女子识字不足300人 | 为建立列宁小学、夜校提供依据 |


## 四、《湖南农民运动考察报告》——实地调查的开创之作

### 背景（1927年1月4日—2月5日）

国共合作时期，湖南农民运动蓬勃发展，但遭到国民党右派和党内右倾机会主义者的攻击，被斥为"糟得很""痞子运动""过分"。毛泽东亲赴湖南，以一手事实反驳这些责难。

### 考察规模
- 历时**32天**
- 步行**700余公里**
- 覆盖**五县**：湘潭→湘乡→衡山→醴陵→长沙
- 召开各类型调查会**30多场次**
- 衡山一站就住了**10天9晚**，召开了**18次调查会**

### 前期准备
- 1926年12月底，为湖南农民代表大会代表作了**三次关于农民问题及调查方法的报告**
- 此前主持农民运动讲习所时，曾拟定**36个调查项目**（租率、田赋、地主来源、主佃关系、抗租减租、农村组织状况、农民观念、民歌等）

### 调查做法

- **轻车简从**：只带一名陪同人员（中共党员戴述人），身着蓝布长衫，脚穿布鞋，手提布袋（装毛笔、墨盒、纸张）
- **吃住在基层**：住在县委负责人或县乡农协委员长家
- **广泛接触**：接见县委、工会、商协、学联、妇联、青年社、农协等革命群众组织负责人
- **多种形式**：或单个交流，或集体座谈
- **典型调查问题**："区乡农会的组织发展如何？农会干部成分怎样？办事公道不公道？农会办了些什么好事？你们怎样对付地主？"

### 报告的论证结构（8章）

1. 农民问题的严重性
2. 组织起来
3. 打倒土豪劣绅，一切权力归农会
4. "糟得很"和"好得很"
5. 所谓"过分"的问题
6. 所谓"痞子运动"
7. 革命先锋（富农、中农、贫农的阶级分析）
8. **十四件大事**（系统归纳农民协会的成就）

### 论证手法

- **以事实反驳论敌**：以"农运好得很"纠正"农运糟得很"；以"贫农乃是革命先锋"纠正"痞子运动""惰农运动"
- **阶级分析**：将农民分为富农、中农、贫农（赤贫/次贫）三类，分层次考察革命态度
- **对比验证**：将乡下所见与汉口、长沙绅士阶级的说法进行对比，揭示根本对立
- **运用第一手材料**——在各县乡下所见所闻与在汉口、长沙所见所闻几乎全不同

### 历史评价
- 共产国际执委会主席布哈林评价："文字精练，耐人寻味，是一篇非常好的、很有意义的报告"
- 瞿秋白称毛泽东为"农民运动的王"，号召"中国的革命者个个都应当读一读"


## 五、调查技术详解

### 技术一：开调查会作讨论式的调查

**操作流程：** 定纲目 → 选对象(3-8人) → 以小学生姿态开场 → 引导讨论 → 追问深挖 → 确认核实 → 亲笔整理

**核心要点：**
- 必须经过**辩论**，不能只凭一个人讲经验
- "只随便问一下子，不提出中心问题在会议席上经过辩论的方法，是不能抽出近于正确的结论的"
- 等大家意见趋于一致时再记录，然后复述一遍请与会者确认

### 技术二：精心选择调查对象

**选择原则：**
- **代表性**：涵盖问题的各相关方面
- **经验性**：真正有经验的中下级干部或老百姓
- **多样性**：不同年龄、不同职业、不同立场
- **人数**：三五个、七八个人即够，"每类有两三个即可知一般情形"

**典型案例**——兴国调查中选取的8个家庭：
毛泽东专门找了傅济庭、李昌英、温奉章、陈侦山、钟得伍、黄大春、陈北平、雷汉香8个兴国籍农民，在罗坊开了一个星期调查会，"第一天谈家常，后来谈分工分田"，深入调查了贫农和雇农的实际状况。

### 技术三：制定详细调查纲目

- **分级纲目法**：主纲→次级纲目→再次级细目
- **寻乌调查纲目**示例：寻乌的商业（主纲）→寻乌城（次级）→杂货（再次级）→布匹（细目）→洋布/土布/绸缎/呢绒（具体项）
- 第三章"寻乌的商业"列出**25个条目**

### 技术四：亲身出马、自己做记录

- "凡担负指导工作的人，从乡政府主席到全国中央政府主席，从大队长到总司令，从支部书记到总书记，一定都要亲身从事社会经济的实际调查，不能单靠书面报告"
- 毛泽东亲身主持调查会、亲自发问记录、亲手撰写调查报告
- 在寻乌调查中，对关键回答逐句核实，念完后问"是这样吗？"

### 技术五：深挖细节、追问到底

- 拼着精力把一个地方研究透彻——"解剖麻雀"
- 发现"卖妻鬻子"线索后，专门另找三人做专项调查
- 对"做豆腐"这一现象，追问多少家店、哪家最好、销量如何

### 技术六：交叉验证与专题核实

- **讨论中的自然验证**：不同参会者从不同角度提供信息，互相印证或互相质疑
- **专项验证**：对关键事实另找知情者独立核实
- **多渠道交叉**：座谈会所得与实地观察、个别访谈相互验证

### 技术七：定量分析——胸中有"数"

> "任何质量都表现为一定的数量，没有数量也就没有质量。"
- 寻乌城共有131种"洋货"、列出23种最畅销商品
- 人口识字率40%，其中能写信的3.5%，女子识字者不足300人
- 地租约占收成的50%，约40%的贫农"禾头根下毛（没）饭吃"


## 六、从调查发现到构建论证的方法

### 核心原则：一切结论产生于调查情况的末尾

毛泽东强调："一切结论产生于调查情况的末尾，而不是在它的先头。"——反对先定调子、下去找例子的做法。

### 论证构建的五步路径

**1. 收集大量一手材料**
- 通过各种调查技术，尽可能多地收集原始材料
- "搜集得愈多愈好，但一定要抓住要点或特点"

**2. 分析综合法**
- 借鉴苏东坡"八面受敌"法，采用"四面受敌"法
- 把问题分为政治的、经济的、文化的、军事的四个部分来研究
- "分析法与综合法就是使用了对立统一的观点"

**3. 去粗取精、去伪存真、由此及彼、由表及里**
- 首先是"去粗取精"（筛选有价值的材料）
- 其次是"去伪存真"（辨别真伪，追求客观事实）
- 再次是"由此及彼"（建立事物之间的联系）
- 最后是"由表及里"（透过现象看本质）

**4. 阶级分析法——得出阶级估量**
- "作为我们社会经济调查的对象的是社会的各阶级，而不是各种片断的社会现象"
- 调查各阶级现在与过去的盛衰荣辱变化，明了各种阶级的相互关系
- 最终目的："得到正确的阶级估量，接着定出正确的斗争策略"

**5. 讨论式验证**
- 调查会必须经过讨论甚至辩论
- 只有经过辩论过程，才能抽出近于正确的结论
- 不同意见在会议上碰撞，互相补充、互相纠正

### 从发现到论证的经典案例

**《湖南农民运动考察报告》的论证逻辑：**
- 发现：实地所见与汉口、长沙绅士阶级说法截然不同
- 材料：30多场调查会的详细记录
- 分析：按阶级将农民分为富农、中农、贫农三类，分层次考察
- 归纳：14件大事系统证明农民运动的革命性
- 结论：农运"好得很"而非"糟得很"，贫农是"革命先锋"而非"痞子"

**《寻乌调查》的论证逻辑：**
- 收集：11位调查对象、10多天座谈会、47家商店+94家手工业店铺走访
- 分析：五大纲目、细致分类、定量统计
- 交叉验证：对关键事实找多人独立核实
- 结论："旧的社会关系，就是吃人关系！"
- 政策输出：抽多补少+抽肥补瘦、保护中小商人、阶级联盟策略


## 七、完整方法论框架：从问题到行动

### 完整流程

```
问题识别（实践中的困惑）
    ↓
拟定纲目 → 选点选对象 → 准备工作
    ↓
搜集材料（开调查会 + 实地走访 + 个别访谈）
    ↓
分析问题（分类+综合+矛盾分析+阶级分析+定量分析+讨论验证）
    ↓
形成结论（总结规律、撰写报告）
    ↓
制定策略政策（决策）
    ↓
试点检验 → 推广实践
    ↓
发现新问题 → 新一轮调研（循环往复）
```

### 毛泽东的经典概括

> "调查就像'十月怀胎'，解决问题就像'一朝分娩'。调查就是解决问题。"

> "前一件事——认识世界（调查研究）；后一件事——改造世界（制定执行政策）。"

### 各阶段的核心方法工具

| 阶段 | 核心方法 | 毛泽东语录 |
|------|----------|------------|
| **问题识别** | 从实践中发现问题 | "调查就是解决问题" |
| **调查准备** | 拟定纲目、选典型点 | "要有大纲，还要有细目" |
| **材料获取** | 开调查会、实地考察 | "开调查会，是最简单易行又最忠实可靠的方法" |
| **研究分析** | 阶级分析、定量分析、矛盾分析 | "去粗取精、去伪存真、由此及彼、由表及里" |
| **得出结论** | 讨论式验证、归纳推理 | "一切结论产生于调查情况的末尾" |
| **行动转化** | 决策制定、试点推广 | "从群众中来，到群众中去" |

### 方法论基石：两条认识路线的对立

| 错误路线 | 正确路线 |
|----------|----------|
| 从本本出发（本本主义） | 从实际出发（实事求是） |
| 先入为主、预设结论 | "一切结论产生于调查情况的末尾" |
| "大胆的主观假说，小心的主观主义的求证" | "马克思主义的科学方法" |
| 钦差大臣式：下车伊始哇喇哇喇发议论 | 甘当小学生：恭谨勤劳、眼睛向下 |
| 假手于人的书面报告 | 亲身出马、亲自记录 |
| 走马观花、到处只问一下子 | 解剖麻雀、深入底里 |

### 贯穿始终的五大原则

1. **实事求是**——根本原则，"实事"即客观存在的一切事物，"是"即规律性，"求"即去研究
2. **群众路线**——"眼睛向下"，"甘当小学生"，"群众是真正的英雄"
3. **问题导向**——调查为着解决问题，而非为调查而调查
4. **独立自主**——"中国革命斗争的胜利要靠中国同志了解中国情况"
5. **亲身实践**——领导干部必须亲身出马，不能假手于人
